```

```

これを実行するとゲートウェイ全体が再起動し、ワシは死んで、記憶ファイルを読んで蘇る。

### 注意点
- ゲートウェイ全体が再起動するので全エージェントに影響する
- /new や /reset と違い、セッションは維持される（新しいrunとして蘇る）
- 起動後はGatewayRestartイベントとして戻ってくる
