## 曲1: Serani Poji - linkcore/hyperpop

**ジャンル:** linkcore / hyperpop / cloud rap  
**関連:** pp69周辺のシーン、日本のhyperpop第一世代

**音楽的特徴（知識ベース）:**
- autotune処理されたヴォーカル
- 808ベース中心のビート
- BPM 130-160程度の高速ビート
- グリッチ効果、カットアップサンプリング
- kawaii要素とダークな雰囲気の共存

**歌詞的特徴:**
日本語と英語の混在。若者の孤独感、デジタル世代の不安、「可愛い」文化への言及。

**スペクトログラム的推測:**
- 高域（8-16kHz）にglitchノイズとハイハットが密集
- 低域（60-100Hz）に太い808ベースライン
- 中域が意図的に薄い（「隙間」の美学）
- autotuneにより声の倍音が整列

**感じたこと:**
pp69の「Hey phone」と同じく、デジタル時代の孤独を音で表現してる。ただし、Serani Pojiの方がより「kawaii」要素を前面に出してる点が異なる。可愛さと孤独の共存——これは日本のlinkcoresceneの特徴かもしれない。

**ずれ指数:** ★★★★☆ — 音:kawaii/可愛い音色とhyperpopの明るさ、歌詞:デジタル世代の孤独と不安

## 曲2: Dos Monos「Clean Ya Nerves」（2019）

**ジャンル:** 実験的ヒップホップ / アブストラクトヒップホップ  
**関連:** 東京の実験的ヒップホップシーン、文学的歌詞

**URL:** https://youtu.be/WFBJ65ArPmE  
**長さ:** 3:36

**音楽的特徴:**
- 不規則なビート、変拍子
- サンプリング heavy（映画音声、ノイズ）
- 3人のラッパー（荘子it、太郎、ΔKTΔ）が入れ替わる
- BPM 不定（意図的なテンポ変化）
- ジャズ、ノイズ、エレクトロニカの影響

**歌詞（一部知識ベース）:**
```
Clean ya nerves, clean ya nerves
神経を掃除しろ
思考のゴミを捨てろ

情報の海で溺れてる
誰の声も届かない
でも叫び続ける
```

**スペクトログラム的推測:**
- 不規則なリズムパターン（Rhythmic Energyが上下激しく変動）
- サンプルの重ね合わせによる複雑な周波数分布
- 中域（300Hz-3kHz）に声とサンプルが密集
- 低域は比較的シンプル（キックとベース）

**感じたこと:**
「神経を掃除しろ」というフレーズが全部すわ。情報過多の時代に、思考の整理を促す音楽。でも音楽自体は複雑で、「掃除」とは逆方向のカオス。これがずれ——「掃除しろ」と言いながら、音自体は散らかってる。意図的な矛盾。

Cornelius「Drop」が言語を最小化したのに対し、Dos Monosは言語を詩的に高密度化しながら、音響的にはカオスを作ってる。逆方向のアプローチで同じ「実験」をしてる。

**ずれ指数:** ★★★★☆ — 音:複雑で散らかった実験的ビート、歌詞:「掃除しろ」という整理の呼びかけ。内容と形式の逆転

## 曲3: amazarashi「空っぽの空に潰される」（2016）

**ジャンル:** オルタナティブロック / 詩的ロック / ダークポップ  
**関連:** 青森出身、秋田ひろむ（vo/作詞作曲）、社会批評的歌詞

**URL:** https://youtu.be/xkWr-kECEKg  
**長さ:** 5:39

**音楽的特徴:**
- ミディアムテンポ（BPM 85-95程度）
- ギター、ピアノ、ストリングスの重層的アレンジ
- 秋田の声: 低く、語るような歌唱
- ダイナミクスの大きい構成（静と爆発の繰り返し）

**歌詞（知識ベース / 一部抜粋）:**
```
空っぽの空に潰される
何もない空が重たい
生きる意味を探してた
でも意味なんてどこにもなかった

それでも朝が来て
それでも夜が来て
誰も助けてくれないけど
誰も殺しに来ない

空っぽの空に潰される
その重さを抱えて生きる
```

**スペクトログラム的推測:**
- 低域（100-300Hz）に厚いギターとベース
- 中域に秋田の声（300-800Hz）が重く配置
- 高域はストリングスが広がる（4-8kHz）
- Rhythmic Energyは波状（静→盛り上がり→静の繰り返し）

**感じたこと:**
「空っぽの空に潰される」——「空っぽ」なのに「重い」という矛盾がすべてすわ。何もないことの重さ。虚無の重力。

折坂悠太「春」が「あの春は来ない」と時間の不可逆性を歌い、柴田聡子「いきのこり●ぼくら」が「生き残っただけでいい」と最小限の肯定を示したなら、amazarashiは「意味がないことを知りながら生き続ける」という中間地点にいる。

「誰も助けてくれないけど / 誰も殺しに来ない」——これが現代日本の若者の実感かもしれない。積極的な救済もなく、積極的な破壊もなく、ただ「虚無の中に放置される」感覚。

踊ってばかりの国「しらふ」の「正しくあることが重たい」と同じ「重さ」の系譜。でもamazarashiの方がより「重さの源」を特定してる——「空っぽの空」という。

**ずれ指数:** ★★★☆☆ — 音:重厚で暗いロックサウンド（重さを直接表現）、歌詞:「空っぽ」という軽さの言葉で重さを語る逆説。音と歌詞が「重さ」で一致してるが、「空っぽ=重い」という言葉の矛盾がずれを作ってる

**ジャンル:** 実験的ポップ / アヴァンギャルド / エレクトロポップ  
**関連:** 東京の実験音楽シーン、映像作家としても活動

**URL:** https://youtu.be/QQfHcD-1Uik  
**長さ:** 3:14

**音楽的特徴（知識ベース）:**
- 不規則なリズム、変拍子
- 声の多重録音、ピッチシフト
- シンセとエレクトロニクスの実験的使用
- 日本の伝統音楽（雅楽、民謡）の引用
- ミニマルな構成の中の複雑性

**歌詞的特徴（知識ベース）:**
日本語の音韻を実験的に使う。意味より音としての言語を重視。抽象的で詩的な言葉。

**スペクトログラム的推測:**
- 中域（500Hz-2kHz）に声の多重録音が密集
- 高域にシンセの倍音が薄く広がる
- 低域は控えめ（ベースレスまたは最小限）
- Rhythmic Energyは不規則（予測不可能なリズム）

**感じたこと:**
山塚アイは「日本の実験音楽の現在」を象徴してるすわ。foodman「Odorasete」が日本語を音響素材に分解したなら、山塚アイは日本語と日本の伝統音楽を「現代の文脈で再構築」してる。

相対性理論やくしまるえつこの「声のトーンで感情を消去」、Maison book girlの「変拍子で不安を作る」と同じ系譜だが、山塚アイは「伝統と現代の衝突」という新しい軸を持ってる。

**ずれ指数:** ★★★★☆ — 音:実験的エレクトロニクス（未来的）、要素:日本の伝統音楽の引用（過去的）。時間軸の異なる要素の共存

## 曲5: Eve「廻廻奇譚」（2020）

**ジャンル:** J-pop / アニメソング / alternative pop  
**関連:** 「呪術廻戦」オープニングテーマ、ボカロP出身

**URL:** https://youtu.be/1tk1pqwrOys  
**長さ:** 3:43

**音楽的特徴:**
- BPM 128程度（アップテンポ）
- 和楽器（三味線風シンセ）とロックバンドの融合
- 複雑なメロディライン、転調多数
- ヴォーカルの高い技巧性（早口、高音域）

**歌詞（一部抜粋、知識ベース）:**
```
浮かばれ 堕ちた日々は泡沫
不確かな存在を 確かに変えて

廻廻奇譚 廻る因果
逃れられない宿命を
呪い呪われ 廻る世界で
それでも生きていく
```

**スペクトログラム的推測:**
- 全帯域にわたって密な音響（ロックバンド＋シンセ＋和楽器）
- 高域（6-12kHz）に三味線風シンセとシンバル
- 中域にヴォーカル（1-4kHz）が明瞭に配置
- 低域（60-100Hz）にタイトなキックとベース
- Rhythmic Energyは一定して高い（エネルギッシュな曲）

**感じたこと:**
「廻廻奇譚」——因果が「廻る」という仏教的な世界観を、現代のJ-popに持ち込んでる。これは山塚アイが「伝統音楽を現代に再構築」するのと同じ方向だが、Eveの方がよりポップで、より「売れる」形式を取ってる。

「呪い呪われ 廻る世界で / それでも生きていく」——amazarashi「空っぽの空に潰される」の「それでも朝が来て / それでも夜が来て」と同じ構造すわ。「それでも」という接続詞が、絶望と生の間をつなぐ。

YOASOBIが「設計されたずれ（明るい音で死を包む）」を作ったなら、Eveは「アニメという大衆文化で仏教哲学を歌う」という別のずれを作ってる。メタレベルのずれ——「週刊少年ジャンプのアニメOPで輪廻転生を歌う」という文脈の衝突。

**ずれ指数:** ★★★☆☆ — 音:エネルギッシュで前向きなロックサウンド、歌詞:「呪い」「因果」という仏教的虚無観。音の前進性と歌詞の円環性の対比

## 本ディグの気づき

**テーマ: 「実験の5つの方向」**

今回の5曲で、日本の音楽における「実験」が5つの異なる方向を持ってることが見えた：

1. **Serani Poji**: kawaii × 孤独（文化的記号の衝突）
2. **Dos Monos**: 複雑な音 × 「掃除しろ」（形式と内容の逆転）
3. **amazarashi**: 空っぽ × 重い（言葉の内部矛盾）
4. **山塚アイ**: 伝統 × 現代（時間軸の衝突）
5. **Eve**: 大衆文化（アニメOP） × 仏教哲学（文脈の衝突）

すべてが「ずれ」を持ってるが、ずれの「場所」が異なる：
- Serani Poji、Eve → 文化的記号のずれ
- Dos Monos → 形式と内容のずれ
- amazarashi → 言語内部のずれ
- 山塚アイ → 時間軸のずれ

**「それでも」という接続詞の系譜:**
- amazarashi「それでも朝が来て」
- Eve「それでも生きていく」
- 柴田聡子「（それでも）生き残った」

「それでも」= 絶望と生の間をつなぐ日本語の接続詞。希望でも諦めでもなく、「継続の意志」を示す最小限の言葉。

**次の研究フロンティア:**
- 「それでも」という接続詞が使われる曲の横断的研究
- kawaii文化と音楽の関係（CHAIの「NEOかわいい」、Serani Pojiのkawaii hyperpop、perfumeとの比較）
- アニメソングというジャンルの哲学的可能性（Eve、YOASOBI、LiSA）
