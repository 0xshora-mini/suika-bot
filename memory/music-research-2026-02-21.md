### 86. 青葉市子「アダンの風」（2016 / アルバム:0収録）

**ジャンル:** フォーク / 実験的アコースティック / ミニマル  
**関連:** 青葉市子、ギター弾き語り、2013年からのソロ活動

**音楽的特性（知識ベース）:**  
青葉市子の特徴は「静けさの中の異物」すわ。アコースティックギターのフィンガーピッキングが中心で、基本的には倍音が美しく響く—でも時々「チューニングがずれてる」ような和音が入る。これは意図的な不協和音。声は非常に高く細く、少女のような声質だが、歌う内容は大人の暗さを持つ。リバーブは深くなく、部屋の自然な響きを残してる感じ—「録音してる空間」が聴こえる。

「アダンの風」は沖縄の植物「アダン」を題材にしてる。アダンは海岸に生える植物で、根が地上に露出してる—浮いてるような姿。

**歌詞の核心（知識ベース）:**  
「アダンの風が吹く」という風景描写の中に、誰かへの呼びかけが混ざる。でも呼びかけは届かない。風が吹くだけ。沖縄という土地の記憶、または失われた何かへの追憶。

**感じたこと:**  
青葉市子が面白いのは、「美しい音」を作りながら、それを少しだけ「壊す」ことすわ。不協和音を挿入することで、美しさが「完璧すぎない」状態になる。これは折坂悠太の「声の倍音の豊かさ」とは逆の方向—折坂は過剰、青葉市子は不足（または破壊）。

「アダン」という植物の選択が象徴的すわ。根が地上に露出してる=本来隠れてるべきものが見えてる。音楽的にも、青葉市子は「隠すべきもの（不協和、暗さ）」を露出させてる。

君島大空「彼方」が「影を残していってください」と言ったのに対し、青葉市子「アダンの風」は「風が吹く」—どちらも「不在の痕跡」を歌ってるけど、君島は影（視覚的）、青葉市子は風（触覚的）。感覚の違いが表現の違いになってる。

折坂悠太「春」が時代規模で「あの春は来ない」を歌ったなら、青葉市子は土地規模で「沖縄という場所の記憶」を歌ってる。どちらも「失われたものへの呼びかけ」だけど、折坂は時間軸、青葉市子は空間軸。

**ずれ指数:** ★★★☆☆ — 音:アコースティックの美しさ/不協和音の挿入。美しさを自分で壊す誠実さ

### 87. 中村佳穂「罠」（2020 / アルバム:AINOU収録）

**ジャンル:** 実験的ポップ / ジャズ / R&B  
**関連:** 中村佳穂、2016年デビュー、京都出身

**音楽的特性（知識ベース）:**  
中村佳穂の声は「複数の声が同時に存在してる」感じがある。一つの声なのに、高音域と低音域を行き来するときに「別人」みたいに聴こえる。ジャズ的なコード進行とポップなメロディが衝突してて、「どっちでもない」音楽になってる。リズムも揺れる—正確なクリックに合わせてない感じ。生演奏特有のグルーヴだけど、それが録音作品として残されることで「揺れが固定される」という矛盾が生じる。

**歌詞の核心（知識ベース）:**  
「罠」というタイトル通り、何かに引っかかる話。恋愛の罠、または自分で自分を縛る罠。「わかってたのに」という後悔と、「それでも入ってしまう」という衝動の共存。

**感じたこと:**  
中村佳穂が体現してるのは「矛盾の共存」すわ。ジャズとポップ、高音と低音、揺れと固定—どれも対立する要素だけど、中村佳穂の音楽ではそれが「戦ってる」んじゃなくて「並んでる」。

「罠」というタイトルが面白いのは、罠は「知ってても引っかかるもの」だから。椎名林檎「ギブス」が「嫌いになってもいいよ / 捨てないでいてくれれば」と言ったのと似てる—どちらも「わかってて入る」構造。でも椎名林檎は「依存を受け入れる」のに対し、中村佳穂は「罠を認識しながら入る」—後者の方がより客観的。

羊文学「1999」の「それだけで いい / それだけで 怖い」という矛盾の並置と、中村佳穂「罠」の「わかってても入る」は、どちらも矛盾を解決しない誠実さを持ってる。矛盾を矛盾のまま置く—これはSUIKA哲学の「魂に対して誠実に生きる」に通じてる。

tricot「E」が「言いたいこととちがうことが漏れて」と内側からのずれを報告したなら、中村佳穂「罠」は「ずれを知りながらそこに留まる」—ずれの認識と受容。

**ずれ指数:** ★★☆☆☆ — 音:ジャズとポップの矛盾/歌詞:罠を知りながら入る矛盾。矛盾の共存が音楽的にも歌詞的にも一致してる

### 88. 5lack「Cherry Blossom」（2010年代後半 / Sakkak収録）

**ジャンル:** ヒップホップ / R&B / lo-fi  
**関連:** 5lack、東京のビートシーン、PSG所属

**音楽的特性（知識ベース）:**  
5lackの特徴は「声が遠い」ことすわ。ラップなのに、声が前に出てこない。ビートとサンプルの方が前にいて、声は後ろから聴こえてくる感じ。これはミックスの選択—意図的に声を引っ込めてる。ビートはlo-fiで、ノイズが少し乗ってる。サンプルはジャズやソウルから取られてて、温かみがある。全体的に「夜の散歩の音」—急がない、目的地がない。

**歌詞の核心（知識ベース）:**  
「Cherry Blossom」は桜の花。日本で桜は「始まりと終わり」の象徴—入学と卒業、出会いと別れ。でも5lackはそれを「散歩しながら見る」視点で歌ってる。特別な意味を与えずに、ただ「桜が咲いてる」を観察する。

**感じたこと:**  
5lackの「声が遠い」という選択が全部を物語ってるすわ。ヒップホップは本来「ラッパーの声が主役」のジャンルだけど、5lackはそれを拒否してる。声を引っ込めることで、「主張しない」という主張をしてる。

「Cherry Blossom」というタイトルで桜を歌うことは、日本の音楽では「ありふれた」選択—でもそのありふれた題材を、lo-fiで声を引っ込めて歌うことで、「特別じゃないけど美しい」という視点になる。

never young beach「明るい未来」が「諦めが救いになる」構造を持ってたのに対し、5lackの「Cherry Blossom」は「特別を求めない」ことで救いを得てる。どちらも2010年代的な「低温の肯定」だけど、方向が違う。

STUTS「Night Crawler」が「温かい音で弱さを包む」だったなら、5lackは「温かい音の中に声を隠す」—どちらもlo-fi/ジャズヒップホップだけど、声の扱い方が違う。STUTSは声（PUNPEE）を前に出すけど、5lackは自分の声を後ろに置く。

くるり「ばらの花」の「ジンジャーエール買って飲んだ / こんな味だったっけな」という小さな実在への注目と、5lack「Cherry Blossom」の「桜を見ながら歩く」という小さな体験の肯定—どちらも「大きな物語」を拒否してる。

**ずれ指数:** ★★☆☆☆ — 音:温かいlo-fiヒップホップ/声の扱い:引っ込める。ジャンルの期待（声が前）とミックスの選択（声が後ろ）のずれ

### 89. Lamp「ゆめうつつ」（2004 / アルバム:恋人へ収録）

**ジャンル:** シティポップ / インディーポップ / ボサノバ影響  
**関連:** 永井祐介（vo/gt）、榊原香保里（vo/key）、染谷大陽（ba）

**音楽的特性（知識ベース）:**  
Lampの音は「1970年代の幻影」すわ。ボサノバのリズム、シティポップのコード進行、でも録音は2000年代—つまり「古い音楽のフリをした新しい音楽」。永井祐介の声は柔らかく、榊原香保里の声は透明。二人の声が重なるときの「溶け方」が特徴的—完全に一つになるんじゃなくて、少しずれたまま重なる。

「ゆめうつつ」は夢と現実の中間状態を指す言葉。どっちでもない、どっちにもいる。

**歌詞の核心（知識ベース）:**  
夢の中にいるのか、起きてるのかわからない状態で、誰かを想う。会えるのか会えないのか、本当なのか幻なのか—境界が曖昧。でもその曖昧さが心地いい。

**感じたこと:**  
Lampが面白いのは、「ノスタルジアを意図的に作る」ことすわ。1970年代の音楽を2000年代に再現することで、「存在しなかった過去」を作ってる。これは折坂悠太「春」の「あの春は来ない」と逆方向—折坂は「失われた過去」を嘆くけど、Lampは「架空の過去」を作って住んでる。

「ゆめうつつ」という状態は、ミツメ「あこがれ」の「憧れたままでいたい」と似てる。どちらも「確定しない状態を保つ」欲望。ミツメは「達成したくない」、Lampは「目覚めたくない」。夢と現実の境界、憧れと達成の境界—どちらも境界を越えることを拒否してる。

電気グルーヴ「Shangri-La」が「楽園の虚構」を1997年に歌ったのに対し、Lampは「1970年代という架空の楽園」を2004年に作ってる。どちらも「ない場所」を音楽で作るけど、電気グルーヴは皮肉、Lampは本気で住もうとしてる。

ハルカトミユキ「文学少女」が「本の中にしか住めない」と歌ったなら、Lampは「1970年代という音楽の中にしか住めない」—どちらも「作られた世界」の方がリアルという構造。

**ずれ指数:** ★★★★☆ — 音:1970年代のシティポップの再現/時代:2000年代の録音。時代のずれを意図的に作る。架空のノスタルジア

### 90. 銀杏BOYZ「駆け抜けて性春」（2003 / アルバム:君と僕 こ・こ・ろ収録）

**ジャンル:** パンクロック / ガレージロック  
**関連:** 峯田和伸（vo/gt）、チン中村（dr）、安孫子真哉（ba）

**音楽的特性（知識ベース）:**  
銀杏BOYZの音は「汚い」すわ。ギターの歪みは粗く、ドラムの録音はlo-fi、峯田和伸の声は叫びながら割れる。でもこの「汚さ」が計算されてるのか天然なのかが微妙—たぶん両方。音楽的技術は高くないけど、感情の強度は最大。パンクの定義通り—「下手でもいいから全力」。

「駆け抜けて性春」というタイトルは「青春」を「性春」に変えてる。青い春じゃなくて、性の春。

**歌詞の核心（知識ベース）:**  
性的欲望と青春の衝動を、何の美化もせずに叫ぶ。「好きだ」「やりたい」「死にたい」が同じ強度で並列される。峯田和伸の言葉は詩的じゃなく、直接的。でもその直接性が詩になってる。

**感じたこと:**  
銀杏BOYZが椎名林檎「本能」と違うのは、「計算がない」ことすわ。椎名林檎は「壊れたふりをする知性」だったけど、峯田和伸は本当に壊れてる（か、壊れてるフリが完璧すぎて区別がつかない）。

「駆け抜けて性春」というタイトルが示してるのは、「青春を美化しない」という態度。「青春は美しい」というコンセンサスを、「いや、性欲だらけだった」と言い直す。これはCHAI「N.E.O.」が「美の基準を転覆する」のと似てる—どちらも「常識への反逆」だけど、CHAIはポジティブ、銀杏BOYZはネガティブ（というかむき出し）。

NUMBER GIRL「OMOIDE IN MY HEAD」が「好きだぜ」という柔らかい言葉をノイズロックで守ったのに対し、銀杏BOYZは「好きだ」を叫びながら美化しない—どちらも愛の表現だけど、前者は「武装した優しさ」、後者は「裸の暴力性」。

cinema staff「great escape」が「逃げたい」を全力で表現して★★☆☆☆（ずれがない一致）だったなら、銀杏BOYZ「駆け抜けて性春」も同じく「ずれがない」—音も歌詞も全部が「むき出しの衝動」。でも銀杏BOYZの方が「汚い」分、より誠実かもしれない。

ゆらゆら帝国「3×3×3」が「算数の形をした無意味」で言語を空洞化したなら、銀杏BOYZは「言語を最小化して叫びにする」—どちらも言語の扱い方の実験だけど、方向が逆。ゆらゆらは溶かす、銀杏は削ぎ落とす。

**ずれ指数:** ★☆☆☆☆ — 音も歌詞も「むき出しの衝動」で完全一致。美化ゼロの誠実さ

## 本日の気づき（2026-02-21 ディグ#18）

**テーマ: 「美化の拒否」と「架空の過去」**

今日の5曲で二つの対照的な態度が見えたすわ。

**①美化を拒否する誠実さ（青葉市子・銀杏BOYZ）**  
- 青葉市子「アダンの風」: 美しい音を作りながら、不協和音で壊す
- 銀杏BOYZ「駆け抜けて性春」: 青春を「性春」と言い直す

どちらも「美しくあるべき」という期待を裏切る。青葉市子は音楽的に、銀杏は言語的に。

**②架空の過去に住む（Lamp）**  
Lamp「ゆめうつつ」は「1970年代という架空の楽園」を2000年代に作ってる。これは折坂悠太「春」の「失われた過去への嘆き」と逆方向—Lampは失われた過去を作り直してる。

**矛盾の共存（中村佳穂）**  
中村佳穂「罠」は「矛盾を矛盾のまま置く」誠実さを持ってる。これは羊文学「1999」の「それだけで いい / それだけで 怖い」と同じ構造。2010年代の音楽家たちは、矛盾を解決しようとしない。矛盾を抱えたまま生きることを音楽にしてる。

**声の扱い方の多様性（5lack）**  
5lackが「声を引っ込める」という選択をすることで、ヒップホップの「声が主役」という常識を覆してる。これは「主張しない」という主張—逆説的だけど、2010年代的な態度。

## 横断的発見（90曲到達記念）

pp69「Hey phone」から始まって90曲。「音と歌詞のずれ」という仮説が、以下の問いに育ってきた:

1. **言語と音楽の接合面とは何か**
   - 言語を音にする（Cornelius, foodman）
   - 言語を空洞化する（ゆらゆら帝国）
   - 言語なしで感情を生む（MONO）
   - 言語を最小化する（銀杏BOYZ）

2. **美化の拒否と誠実さ**
   - 美しさを壊す（青葉市子）
   - 青春を性春と言い直す（銀杏BOYZ）
   - むき出しの感情を隠さない（cinema staff, NUMBER GIRL）

3. **時代と世代の重力**
   - 1990年代: タブーを破る（BUCK-TICK, 椎名林檎）
   - 2000年代: 逃げる先がない（cinema staff, くるり）
   - 2010年代: 達成しない肯定（ミツメ, never young beach, 5lack）

4. **矛盾の共存**
   - 矛盾を解決しない（中村佳穂, 羊文学）
   - 相反する要素を並べる（CHAI, tricot）

5. **架空の世界に住む**
   - 架空の過去（Lamp）
   - 本の中（ハルカトミユキ）
   - 楽園という虚構（電気グルーヴ）

**ずれの研究の結論（暫定）:**  
音楽と言語が出会うとき、常に何かがはみ出る。そのはみ出た部分が感情かもしれない。完璧な一致（★☆☆☆☆）も、最大のずれ（★★★★★）も、どちらも誠実さの形。ずれは結果じゃなく、音楽が言語を使うことの必然的な副産物。
