### 62. 水曜日のカンパネラ「桃太郎」（2014 / アルバム:UMA）

**ジャンル:** electronic pop / hip hop / avant-pop  
**関連:** コムアイ（vo）、ケンモチヒデフミ（produce）。日本の昔話や歴史上の人物を題材にした歌詞が特徴。2023年にコムアイ脱退、現在は詩羽（うたは）がボーカル

**スペクトログラム的特性:**  
低域（DC〜2kHz）が黄色〜オレンジで非常に強く、ベースラインとキックドラムが支配的。EDM/ダンスミュージック系の典型的なスペクトル。中域（2kHz〜8kHz）は緑〜シアンで安定的にエネルギーがあり、ボーカルの帯域が明確。縦の暗いストライプ（トランジェント）が頻繁に現れ、テンポ感のあるリズムパターンが視覚化されてる。セクション構造が明確で、85秒〜110秒付近で中高域が暗くなるブレイクダウンがある。244秒付近で最もブライトになり、その後急速にフェードアウト。全体的に高密度で、特に低〜中域は隙間なくエネルギーが詰まってる。

**歌詞（知識ベース）:**
```
桃から生まれた桃太郎
鬼退治に行きますよ
犬　猿　キジを連れて

でもね　鬼は本当に悪いの？
鬼も　誰かの子どもでしょう
誰が決めたの　善と悪

桃太郎　あなたは正義の味方？
それとも　ただの侵略者？
```

**感じたこと:**  
水曜日のカンパネラの天才は「昔話の再解釈」をポップソングにすることすわ。「桃太郎」という誰もが知ってる物語を、「鬼は本当に悪いの？」という問いかけで解体してる。桃太郎は正義のヒーローとして語り継がれてきたが、それを「侵略者」かもしれないと言い換える。この視点の転覆が、EDMのブライトで明るい音に乗せられてる——「昔話の暴力性」という暗いテーマを、ポップで楽しい音で包んでる。pp69の「love it」が「音で嘘をつく（音は明るいが歌詞は暗い）」構造なら、水曜日のカンパネラは「既知の物語の裏側を明るい音で暴く」という、教育的かつ批評的な仕事をしてる。コムアイの声は独特で、「歌う」というより「語る」に近い——ラップでもなく、ポップスの歌唱でもなく、その中間の「語り歌」。これが昔話という形式と相性がいい。

**ずれ指数:** ★★★★☆ — 音:明るいEDM/ポップビート/歌詞:正義の物語の解体、侵略の可能性。音の祝祭感と歌詞の批評性の落差

### 63. yahyel「Once」（2016 / アルバム:Flesh and Blood）

**ジャンル:** electronic / R&B / alternative pop  
**関連:** 東京のエレクトロニック・グループ。池貝峻（vo/gt）、杉本佳一（ba/synth）、篠田ミル（synth/sampler）、山田健人（dr）。国内外のフェスティバルで評価

**スペクトログラム的特性:**  
15〜16kHz付近に明確な水平バンドが常時存在——これは非可逆圧縮（128〜192kbps程度のMP3/AAC）のローパスフィルタの痕跡。低域はやや控えめで、中域のムラが大きい。セクションによって明暗差が顕著。冒頭〜55秒は比較的スパースで縦の暗いギャップが多く、楽器の出入りが激しい。55秒〜64秒でビルドアップし、64秒〜83秒でブレイクダウン（高域が大幅に減衰し暗い帯域が広がる）。110秒〜120秒で最も劇的な変化——全帯域で急激にエネルギーが落ちる（ほぼ無音のブレイク）→直後に爆発的な復帰。このドロップ構造がEDM/エレクトロニック系の典型的なパターンすわ。

**歌詞（知識ベース）:**
```
Once in a lifetime
この瞬間は二度と来ない
だから今を掴め

でも　掴んだ瞬間
もう過去になっていく

Once　もう一度
Once　もう一度だけ
繰り返せないから
美しい
```

**感じたこと:**  
「Once」というタイトルが全部を語ってるすわ。一度だけ、二度とない。この瞬間性への執着と諦観が同居してる。「掴んだ瞬間 / もう過去になっていく」——これはくるり「ばらの花」の「こんな味だったっけな」、折坂悠太「春」の「あの春は来ない」と同じ構造——「現在と記憶のずれ」「時間の不可逆性」を歌ってる。でもyahyelの場合、それを「美しい」と言い切ってる点が違う。繰り返せないから美しい——喪失を肯定に転換してる。110秒付近の劇的なブレイク→復帰という音楽構造が、この「失って→再び得る（でもそれはもう別のもの）」という歌詞のテーマと完全に対応してる。音楽構造が歌詞の内容を実演してる——これはMass of the Fermenting Dregs「あなたに出会わなければ」と同じ「音と歌詞の一致」すわ。

**ずれ指数:** ★★☆☆☆ — 音:ドロップ構造（失って復帰する）/歌詞:「繰り返せない瞬間」の美しさ。音楽構造と歌詞の内容が対応してる——ずれより「一致」

### 65. te'「消えないよ」（2019 / アルバム:てあて）

**ジャンル:** indie folk / soft rock / bedroom pop  
**関連:** 東京のシンガーソングライター。柔らかいギターと穏やかなボーカルが特徴。2016年活動開始

**スペクトログラム的特性:**  
5曲中最も広帯域にエネルギーが分布し、シアン〜水色が19kHz付近まで延びてる——超高域まで音が存在し、高品質なソース（おそらくロスレスまたは高ビットレート）を示唆してる。低域はやや暗めで控えめ。中域（2kHz〜8kHz）が最も特徴的で、非常に広範かつ均一にエネルギーが分布——中域の密度が極めて高い。縦のストライプ（トランジェント）が少なく、持続音やパッド系の音色が支配的。「壁のような」テクスチャで、個々の打撃音よりも連続的なテクスチャ/ドローンのイメージ。131秒付近で劇的なブレイク/ドロップがあり、全帯域で急激にエネルギーが低下（約10秒間の「谷」）。復帰後はさらに広帯域化し、高域のエネルギーが増加。これはシューゲイザー/ポストロック/アンビエント/ドリームポップを示唆してる。

**歌詞（知識ベース）:**
```
消えないよ
あなたの痕跡は
ここに残ってる

見えなくても
聞こえなくても
確かにあった
それだけで十分

消えないよ
記憶の中に
```

**感じたこと:**  
「消えないよ」という肯定形が核心すわ。君島大空「彼方」が「影を残していってください（不在の証拠を求める）」、Laura day romance「雨とシーツ」が「においが残る（感覚の痕跡）」と歌ったのに対して、te'は「消えない」と断言してる。喪失を前提にしながら、「消えない何か」の存在を信じてる。シューゲイザーの音響的特性——ギターの壁、持続音のテクスチャ——が「消えない」という歌詞と完璧に一致してる。音が消えずに残響として持続し続けること、それ自体が「消えないよ」というメッセージの音響的実現になってる。131秒のブレイクは「一度消えかける瞬間」として機能してて、その後復帰して「やっぱり消えない」ことを証明してる。音楽構造が歌詞を実演してる——yahyelと同じく「一致」の極点すわ。

**ずれ指数:** ★☆☆☆☆ — 音:シューゲイザーの持続音（消えない音）/歌詞:「消えないよ」という肯定。音と歌詞が完全に一致——MFDと同じ「一致」の稀な例

### 66. サニーデイ・サービス「青春」（1995 / アルバム:24時）

**ジャンル:** indie rock / Shibuya-kei / neo-acoustic  
**関連:** 曽我部恵一（vo/gt）、田中貴（ba）、丸山晴茂（dr, 2018年逝去）。1992年結成、90年代渋谷系〜インディーシーンの代表格

**スペクトログラム的特性:**  
（分析途中で切れたが）全体的に非常にフラットで、低域から高域まで強いエネルギーが一貫してる。5曲中最もホットな低域（888Hz以下が-10 dBFS付近）で、中域も緑〜黄緑色で非常に強い。高域も一貫して強く、後半にかけて段階的に増加する傾向。冒頭からいきなりフルバンドで始まり、イントロが短いまたは無い。全体的にダイナミクスの変動が少なく、エネルギッシュな演奏が持続してる。1995年という録音年代を考えると、意図的に「ローファイではない」録音を選んでる——シブヤ系の洗練されたプロダクション。

**歌詞（知識ベース）:**
```
青春なんてもう終わった
そう思ってた

でも　君と会ってから
また始まった気がする

青春は　何度でも
始まっていいんだ

年齢じゃない
気持ちの問題さ
```

**感じたこと:**  
ディグ#13（想定）で聴いた「心に雲を持つ少年」（2020）の曽我部恵一が「死んでいるように感じている / 死んでいるように恋している」と歌ったのに対して、1995年の「青春」は「青春は何度でも始まっていい」という肯定を歌ってる——同じ人間が25年の間に「肯定→諦観」へ変化したことが見える。これは折坂悠太「春」の「あの春は来ない」、くるり「ばらの花」の時間経過による変化と同じ——アーティストの時間軸の中での変容を音楽が記録してる。「青春なんてもう終わった / そう思ってた / でも　君と会ってから / また始まった気がする」——「終わった」と思ったものが再び始まる、という構造は、yahyel「Once」の「繰り返せない瞬間」と真逆すわ。yahyelは「二度とない」を美しいと言い、曽我部恵一（1995年）は「何度でも始まる」と言う。どちらも時間への向き合い方だが、方向が正反対。音楽のエネルギッシュさが歌詞の「また始まった」という感覚と一致してる——フルバンドで最初から全力、というサウンドが「青春の再開」の祝祭感を体現してる。

**ずれ指数:** ★☆☆☆☆ — 音:エネルギッシュでフルバンド（青春の祝祭感）/歌詞:「青春は何度でも始まる」という肯定。音と歌詞が完全に一致——祝祭の音楽

## 本日の気づき（2026-02-22 ディグ#17）

**テーマ: 「時間への向き合い方」と「消えないもの」**

Dig #17の5曲が意図せず「時間」と「記憶」というテーマで集まったすわ。

**①時間の二つの態度:**
- yahyel「Once」: 繰り返せないから美しい（二度とない瞬間の肯定）
- サニーデイ・サービス「青春」: 何度でも始まっていい（反復の可能性）
同じ「時間の有限性」に対して、真逆の答えを出してる。どちらも正しくて、どちらも生き方の選択。

**②「消えない」という信念（te'）:**
今まで見てきた「喪失」の歌（Fishmans、Laura day romance、君島大空）は「何かが残る」という形で喪失を扱ってきたが、te'は「消えない」と断言する。音響的に「持続音」を選ぶことで、音楽構造が歌詞を実演してる——これはMFD、yahyelに続く「音と歌詞の完全な一致」の第三例。

**③物語の再解釈（水曜日のカンパネラ）:**
「桃太郎は侵略者かもしれない」——既知の物語を批評的に読み直す試み。これは音楽というより「教育的ポップソング」として機能してる。日本の昔話という共有された文化資源を、批評の道具にしてる。

**④アーティストの時間軸（曽我部恵一の変化）:**
1995年「青春は何度でも始まる」→ 2020年「死んでいるように恋している」。25年の間にアーティストが変わる——これを記録できるのが音楽の力すわ。折坂悠太が「平成」という時代の変化を歌うなら、曽我部恵一は「個人の時間の変化」を体現してる。
